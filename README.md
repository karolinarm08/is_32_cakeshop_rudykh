Інтернет-магазин кондитерської "Ruby" (ІС-32)

Опис проекту:
Проект є веб-системою "Інтернет-магазин кондитерської", яка дозволяє клієнтам здійснювати онлайн-замовлення продукції, а адміністраторам - управляти товарами та замовленнями.

Web-застосунок інтернет-кондитерської з повним циклом роботи користувача: реєстрація, перегляд каталогу, кошик, оформлення замовлення, історія замовлень та адмін-панель для управління товарами і замовленнями.

Система розгорнута на хостингу:
https://cakeshop.great-site.net/index.html

Frontend - HTML/CSS/JS  
Backend - PHP  
Database - MySQL  

Ролі та розподіл завдань:
Ролі розподілені відповідно до основних бізнес-процесів та архітектури SOLID:

Лємаєва О.М. - Full-stack розробник

Рудих К.О. - Full-stack розробник

Філіпович Д.О. - Тестувальник (QA)


Завершено:
інтеграція frontend + backend + БД
налаштоване керування залежностями
всі компоненти працюють разом, реалізуючи функціональні вимоги

Функціональні вимоги виконані повністю:
реєстрація / вхід
перегляд каталогу
перегляд картки товару
додавання в кошик
оформлення замовлення з адресою доставки
історія замовлень
адмін може додавати/редагувати/видаляти товари
адмін може змінювати статуси замовлень

Нефункціональні вимоги також виконані:
≤5 кліків до замовлення - забезпечено
≤3 сек завантаження - оптимізовані зображення, легкий фронтенд
Захист даних - PHP sessions, парольний hash, SSL/TLS
Адаптивність - працює на ПК, планшеті, смартфоні
Крос-браузерність - Chrome, Safari, Edge
100+ одночасних користувачів - InfinityFree + легка архітектура
Масштабованість - розширювана БД, модульні файли
500 товарів - таблиця products підтримує
≤200 MB RAM у браузері - оптимізація сторінок і ресурсів
